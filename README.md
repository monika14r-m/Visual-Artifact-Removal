# Visual Artifact Removal

A Python-based image restoration toolkit that removes visual artifacts such as:

- Watermarks
- Compression artifacts
- Minor image defects

## Features

- OpenCV-based preprocessing
- Inpainting restoration
- Batch image processing
- Automated testing

## Project Structure

src/
tests/

## Installation

pip install -r requirements.txt

## Usage

python remove_watermark.py input.jpg output.jpg
